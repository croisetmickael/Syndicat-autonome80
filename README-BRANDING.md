## 📱 Où les couleurs sont utilisées

| Élément | Couleur | Usage |
|---------|---------|-------|
| Boutons primaires | Bleu #0052A4 | Actions principales (Envoyer, Valider) |
| Boutons secondaires | Rouge #ED1C24 | Actions secondaires (Annuler, Modifier) |
| En-têtes | Bleu foncé #003D82 | Titres, barres de navigation |
| Bannière défilante | Bleu #0052A4 | Fond des actualités |
| Statut OK | Bleu #0052A4 | Messages validés, messages traités |
| Statut Erreur | Rouge #ED1C24 | Erreurs, alertes |
| Texte | Gris #666666 | Contenu lisible |

---

## 🔧 Comment modifier les couleurs

### Dans `src/App.css` :

```css
:root {
  --color-primary: #0052A4;        /* Bleu - Boutons, accents */
  --color-secondary: #ED1C24;      /* Rouge - Alertes, secondaire */
  --color-accent: #003D82;         /* Bleu foncé - En-têtes */
  --color-success: #0052A4;        /* Pour succès */
  --color-danger: #ED1C24;         /* Pour erreurs */
}
```

Changez les valeurs HEX et redéployez sur Vercel ! 🚀
